# Data Analysis Portfolio

Welcome to my Data Analysis Portfolio.

This repository brings together personal and course-based projects focused on data analysis, business intelligence, and data visualization.

Each project was developed to explore real-world questions, solve problems, and demonstrate how data can be used to support objectives and improve decision-making. The projects may involve different stages of the data analysis process, including data collection, cleaning, transformation, exploration, analysis, visualization, and insight generation.

## What you will find here

- Personal data analysis projects
- Course-based projects and practical exercises
- Business-focused analyses
- Data cleaning and transformation processes
- Exploratory data analysis
- Dashboards and data visualizations
- SQL queries and database exercises
- Insights and recommendations based on data
